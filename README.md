Heart Disease prediction using ML

📌 Overview

This project predicts heart disease using Machine Learning and also performs clustering to identify patient risk groups.

⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

Models Used

* Logistic Regression
* Random Forest
* K-Means Clustering
* PCA

📊 Features

* Data preprocessing
* Feature importance analysis
* Visualization
* Clustering with silhouette score

▶️ How to Run

```bash
pip install -r requirements.txt
python src/heart_disease_analysis.py
```
